# audioMotion demo

> **audioMotion** is a high-resolution real-time audio spectrum analyzer and full-featured music player written in JavaScript.

* [**LIVE DEMO**](https://demo.audiomotion.me)
* [Documentation](https://audiomotion.me)
* [GitHub project](https://github.com/hvianna/audioMotion.js)
